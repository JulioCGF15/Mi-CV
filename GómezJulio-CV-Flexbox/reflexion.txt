En esta actividad aprendí que Flexbox es un modelo de diseño en CSS que permite organizar elementos dentro de un contenedor de forma flexible y adaptable. Comprendí cómo funciona la propiedad display: flex y cómo se pueden distribuir los elementos en filas o columnas según sea necesario. La propiedad que me pareció más útil fue flex, especialmente flex: 0 0 300px para fijar el ancho del sidebar y flex: 1 para que el contenido principal ocupe el espacio restante automáticamente. Lo más difícil fue entender cómo el espacio se reparte de manera automática y cómo influye el comportamiento responsive al cambiar el tamaño de la pantalla. Gracias a esta actividad ahora entiendo mejor cómo estructurar una página web limpia, organizada y adaptable a diferentes dispositivos.
